# Terroir
An opinionated extension of Bedrock.

## Requirements

- [Bedrock](https://github.com/roots/bedrock)
- [Composer](https://getcomposer.org/download)

## Installation

Install via Composer:

```bash
$ composer require codilechasseur/terroir
```
